# EnvironmentTagger

## Overview
Tags environmental elements in scenes for consistency and reference.

## Primary Function
Metadata tagging of environmental elements.

## Input Type
- Video frames
- Scene descriptions
- Reference images

## Output Type
- Tagged metadata
- Environment catalogs
- Continuity references

## Recommended Tech Stack
- Python
- Google Cloud Vision API
- Firebase
- Gemini API

## Integration Points
- TimelineAssembler
- FormatNormalizer
- ContinuityTracker
